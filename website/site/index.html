Parental controls for hotspot rules.

SpotChecker helps parents enforce hotspot, Wi‑Fi, mobile data and app-lock rules with automations, Rules Enforcement Schedule, and parent-approved extra time.

What SpotChecker Does

  • Pair child phones with a parent dashboard and manage settings per child.
  • Apply rules for Lock Apps, Turn Hotspot off, Turn Wi‑Fi Off, and Turn Mobile Data Off.
  • Set a Rules Enforcement Schedule, or enforce rules at all times.
  • Guide setup for Screen Time permission, Screen Time password, and prevent app deletions.
  • Use automation guidance to run checks at multiple times, triggers, and battery levels.
  • Child can request extra time from lock screen; parent can approve/deny from child settings.
  • Push notifications alert parent when extra time is requested.

Frequently Asked Questions

  • Does it require Apple Shortcuts? Yes. SpotChecker uses automations to fetch policy and enforce actions reliably.
  • Can I lock apps? Yes. Configure Lock Apps and use the Screen Time setup checklist on the child phone.
  • Can my child request more time? Yes. Requests appear in parent child settings and can be approved or denied.
  • Does schedule have to be on? No. If schedule is off, protections can enforce at all times.
