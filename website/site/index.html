Parental controls that hold up in real life.

SpotChecker helps parents set iPhone rules for apps, hotspot, Wi-Fi, and mobile data, then keep those rules active with guided setup and reliable automation checks.

What You Can Control

  • Per-child controls: Manage each child phone independently from one dashboard.
  • App locking: Choose apps to lock, including critical setup protections.
  • Connectivity rules: Enforce hotspot, Wi-Fi, and mobile data restrictions.
  • Rules schedule: Apply restrictions on a schedule, or keep them always on.
  • Extra time workflow: Child requests more time, parent approves or denies in one tap.
  • Setup hardening: Guided Screen Time password and deletion-protection steps.

Frequently Asked Questions

  • Does SpotChecker require Shortcuts? Yes. Automations are used to apply and refresh enforcement.
  • Can I run it without a schedule? Yes. Turn schedule off to keep protections active all day.
  • Can a child request more time? Yes. Requests appear in parent child settings for approval or denial.
  • Can this help prevent tampering? Yes. Setup includes Screen Time password and deletion-restriction guidance.
