SpotChecker

A lightweight iPhone setup to help parents enforce hotspot rules via Shortcuts + optional Screen Time locking.

What it does

  • Parent enrolls a child device and configures rules + an enforcement schedule.
  • Child installs the Shortcut and enables automations to regularly “check status”.
  • The Shortcut pulls policy from the backend and applies actions (hotspot / Wi‑Fi / mobile data etc.).
