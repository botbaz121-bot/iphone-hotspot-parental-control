Quick troubleshooting

  • Policy not applying? Ensure the child phone has the Shortcut installed and automations enabled.
  • Pairing code invalid/expired? Generate a new pairing code from the parent phone and try again.
  • No internet? The Shortcut can use the last cached policy if previously fetched.
