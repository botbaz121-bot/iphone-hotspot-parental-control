Support

Quick help for the most common setup and enforcement issues.

Quick troubleshooting

  • Rules are not applying? Confirm the child phone has the SpotChecker Shortcut installed and all required automations enabled.
  • No extra-time approval on parent phone? Refresh status, then open parent child settings and check the Extra Time section.
  • Screen Time setup looks incomplete? Complete setup in order: permissions, Screen Time password, deletion restrictions, then lock steps.
  • Pairing code failed or expired? In parent child settings, tap View Pairing Code and use the newly generated code.
  • Updates are delayed? Verify internet on both phones. Policy checks and approvals require connectivity.
