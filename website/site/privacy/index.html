Privacy Policy

Last updated: 2026-02-20

Summary

SpotChecker helps parents manage child phone rules. We collect only the data needed to pair devices, deliver enforcement updates, and show activity history.

Data we collect

  • Account data: Sign in with Apple identifier used to create and secure a parent account.
  • Pairing data: device identifier and pairing credentials used to connect parent and child devices.
  • Rule activity: timestamps and outcome data for enforcement checks and extra-time actions.
  • Notification token: parent push token used for extra-time request alerts.

Data we do not collect

  • We do not access message content, photos, contacts, browsing content, or app content.
  • We do not collect precise location for SpotChecker policy logic.

How we use data

  • To link parent and child devices securely.
  • To return the current rule policy when child automations check in.
  • To show recent activity and extra-time actions in the parent dashboard.
  • To send parent notifications when extra time is requested.

Sharing

We do not sell personal data. Data is shared only with infrastructure providers required to operate SpotChecker.

Contact

If you have questions, email support@spotchecker.app.
